# MIDI钢琴实时五线可视化项目 - Verification Checklist

## Core Functionality
- [ ] MIDI输入设备可以被检测和选择
- [ ] 音符on/off事件可以被正确捕获
- [ ] 力度数据可以被正确读取
- [ ] 高低音双线五线谱（大谱表）正确显示
- [ ] 高音谱号和低音谱号位置准确
- [ ] 音符位置严格对应标准五线谱线/间
- [ ] C4在高音谱表下加一线位置正确
- [ ] 横向坐标对应真实弹奏时间戳
- [ ] 音符宽度对应真实按住时长
- [ ] 力度越重，音符越实越深
- [ ] 力度越轻，音符越透明
- [ ] 时差≤60ms的音符被识别为和弦
- [ ] 和弦音符X坐标对齐，垂直叠放
- [ ] 时差>60ms的音符正常横向错开
- [ ] 画面自动左滚动，保持当前演奏在可视区域
- [ ] 节拍辅助线可以开关
- [ ] 节拍辅助线仅作参考，不吸附音符
